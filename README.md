# deep_learning
Coursework for the [Deep Learning Lecture](http://uvadlc.github.io/) of the Master Artificial Intelligence at the University of Amsterdam in 2018/19. The implementations are based on PyTorch. Assignment sheets can be found [here](https://github.com/uvadlc/uvadlc_practicals_2019). 

Topics were amongs others:

Assignment 1:
- Multilayer Perceptron
- Backpropagation
- Batch Normalization
- Convolutional Neural Networks

Assignment 2:
- Vanilla Recurrent Neural Networks
- Optimization Methods (Stochastic Gradient Descent, ADAM, RSMProb) 
- Long-Short Term Network 
- Recurrent Neural Networks as Generative Models (Example: Text generation using a Long-Short Term Network)
- Graph Neural Networks


Assignment 3:
- Variational Auto Encoders
- Generative Adversarial Networks
- Generative Normalizing Flows
